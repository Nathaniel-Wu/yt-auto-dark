What's new in version 3.0?

Settings
YtAutoDark keeps its default behavior, and gets some settings to adjust the way it's triggered.
Prefers-color-scheme
If active, YtAutoDark follows browser/os system-level color scheme if browser supports it.
Time based toggle
If active, YtAutoDark will only toggle dark theme before and after the configured hours.
Synced
YtAutoDark uses synced storage.
Settings should be shared across all instances of that browser that the user is logged into.
If synced storage is not available, it should automatically fallback to local storage.
Install/update page
This page will allow users to be notified of new major versions.
